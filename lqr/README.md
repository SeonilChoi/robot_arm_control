# LQR 제어 (Linear Quadratic Regulator)

어떤 **평형점** $z_e = [q_e^T,\,0^T]^T$ (속도 $\dot{q}_e = 0$) 주변에서 로봇 동역학을 **선형화**한 뒤, **LQR**로 이득 $K$ 를 구해 $\tau = \tau_e - K(z - z_e)$ 형태로 제어하는 예제입니다.

---

## 이 제어기를 쓰는 이유

- **선형 제어의 이점**: 비선형 로봇을 한 평형점 근처에서 **선형 모델** $\dot{z} = A z + B u$ 로 근사하면, 이론이 잘 정립된 **선형 제어**(예: LQR)를 쓸 수 있습니다.
- **LQR의 의미**: “상태와 입력이 얼마나 벗어나는지”를 2차 비용(가중치 $Q$, $R$)으로 두고, 그 비용을 최소화하는 **선형 상태 피드백** $u = -K(z - z_e)$ 를 구합니다. **Riccati 방정식** 한 번 풀면 $K$가 나와 계산이 가벼워 실시간에 적합합니다.
- **적용 범위**: 평형점 **근처**에서만 선형 근사가 유효하므로, 초기 상태나 목표가 평형에서 너무 멀면 성능이 떨어질 수 있습니다. 이 실험에서는 평형 근처 초기조건을 사용합니다.

---

## 1. 수식 정리

### 상태와 평형

- $z = [q^T,\, \dot{q}^T]^T \in \mathbb{R}^6$
- 평형에서 토크는 $\tau_e = G(q_e)$ (중력과 균형)
- 보조 입력을 $u = \tau - \tau_e$ 로 두면 선형화 모델은 다음과 같습니다.

$$
\dot{z} = A\,z + B\,u
$$

### Riccati 방정식과 이득

비용을 최소화하는 이득 $K$ 는 다음 **Riccati 방정식**의 해 $P$ 로부터 구합니다.

$$
A^T P + P\,A - P\,B\,R^{-1}B^T P + Q = 0
$$

이득과 제어 입력은 다음과 같습니다.

$$
K = R^{-1}B^T P
$$

$$
u = -K\,(z - z_e)
$$

따라서 실제 토크는 다음과 같습니다.

$$
\tau = \tau_e + u = G(q_e) - K\,(z - z_e)
$$

- $Q$: 상태 오차 가중치 (크면 추종이 빨라짐)  
- $R$: 입력 가중치 (크면 토크를 아끼지만 반응이 느려짐)

### 선형화

$f(z,\tau) = [\dot{q}^T,\; (M^{-1}(\tau - C - G))^T]^T$ 에 대해 평형점 $(z_e,\,\tau_e)$ 에서 자코비안을 구합니다.

$$
A = \frac{\partial f}{\partial z} \bigg|_{(z_e,\,\tau_e)}
$$

$$
B = \frac{\partial f}{\partial \tau} \bigg|_{(z_e,\,\tau_e)} = \begin{bmatrix} 0 \\ M^{-1}(q_e) \end{bmatrix}
$$

이 예제에서는 $A$, $B$ 를 **수치 미분**으로 계산합니다.

## 2. 수식–코드 매칭

| 수식 | 코드 |
|------|------|
| $A$, $B$ 선형화 | `dynamics.py`: `linearize(q_e)` |
| CARE $A^T P + PA - P B R^{-1} B^T P + Q = 0$ | `run.py`: `solve_care(A, B, Q, R)` (해밀토니안 고유벡터로 $P$ 계산) |
| $K = R^{-1} B^T P$ | `run.py`: `K_lqr = Rinv @ B.T @ P` |
| $u = -K(z - z_e)$ | `run.py`: `u = -K_lqr @ (z - Z_E)` |
| $\tau = \tau_e + u$ | `run.py`: `tau = TAU_E + u` |

## 4. 입·출력, 제약, 초기조건

| 구분 | 내용 |
|------|------|
| **입력** | 현재 상태 $z = [q^T,\, \dot{q}^T]^T$. 목표는 평형점 $z_e$ 로 고정. |
| **출력** | $\tau = \tau_e - K(z - z_e)$ |
| **제약** | 토크 제한 $\|\tau_i\| \le 50$ N·m. 선형화는 평형 근처에서만 유효. |
| **초기조건** | $q(0)=[0.5,\,0.35,\,0.25]^T$, $\dot{q}(0)=[0.1,\,0,\,0]^T$. 평형 $q_e=[0.4,\,0.3,\,0.2]^T$. |

---

## 5. 외란 실험

$t \in [1.5,\,2.5]$ 초 동안 $\tau_{dist} = [5,\,-2,\,1]^T$ N·m 를 가합니다.

LQR가 외란에 대해 얼마나 빠르게 다시 평형으로 끌어들이는지, 무외란 궤적과 비교할 수 있습니다.
