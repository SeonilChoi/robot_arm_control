# MPC 제어 (Model Predictive Control)

**MPC**는 “앞으로 몇 스텝까지”를 **선형 모델**로 예측하고, 그 구간에 대한 비용을 최소화하는 **입력 열**을 구한 뒤, **그중 첫 입력만** 적용하고, 다음 스텝에서 같은 최적화를 다시 푸는 **재ceding horizon** 방식입니다.  
이론적으로는 제약 없는 **QP**(2차 계획) 한 번으로 한 스텝의 입력을 구합니다.

**이 폴더의 구현 참고:**  
목표 $q_r = [0.5,\,0.4,\,0.3]$ 에서 선형화한 $A$ 가 **개방루프 불안정**(양의 고유값)이라, N스텝 예측 궤적이 폭발하여 QP 기반 MPC가 비정상 동작했습니다.  
그래서 **같은 $(A,B,Q,R)$ 로 LQR 이득 $K$** 를 구하고, **적분 보정**을 더해 다음 형태로 목표 추종을 구현했습니다.

$$
\tau = G(q_r) + u, \qquad u = -K\,(z - z_r) - K_i \int (q - q_r)\, dt
$$

(LQR은 불안정한 $A$ 도 폐루프에서 안정화합니다.)  
아래 수식은 “진짜” MPC 이론을 정리한 것이고, **코드는 위 LQR+적분**으로 동작합니다.

---

## 이 제어기를 쓰는 이유 (MPC 이론)

- **미래를 고려한 제어**: 현재 입력만 쓰는 게 아니라, “앞으로 N 스텝 동안 상태가 어떻게 변할지”를 모델로 예측하고, **그 궤적 전체**에 대한 비용(목표와의 차이 + 입력 크기)을 최소화합니다. 한 스텝만 보는 제어보다 더 나은 궤적을 기대할 수 있습니다.
- **재ceding horizon**: 매 스텝마다 현재 상태를 기준으로 같은 최적화를 다시 풉니다. 예측과 실제가 어긋나도(모델 오차, 외란) 다음 스텝에서 보정됩니다.
- **제약 추가 가능**: 나중에 토크 한계나 관절 한계를 부등식 제약으로 넣으면, 같은 MPC 틀 안에서 제약을 만족하는 입력을 구할 수 있습니다. (이 예제는 제약 없는 QP만 이론적으로 다룹니다.)

## 1. 수식 정리 (MPC 이론)

### 이산시간 모델

상태 $z_k = [q^T,\, \dot{q}^T]^T$, 입력 $u_k = \tau_k - G(q_r)$ 로 두고 이산화합니다.

$$
z_{k+1} = A_d\, z_k + B_d\, u_k
$$

- $A_d = I + A\,\Delta t$, $B_d = B\,\Delta t$ (선형화 $A$, $B$ 기준)

### 비용 함수

목표 $z_r$ 에 가깝게, 입력은 작게 쓰도록 비용을 둡니다.

$$
J = \sum_{k=0}^{N-1} \Big( \|z_k - z_r\|_Q^2 + \|u_k\|_R^2 \Big) + \|z_N - z_r\|_{Q_N}^2
$$

- $z_r$: 목표 상태  
- $Q,\, R,\, Q_N$: 추종 강도, 입력 크기, 종단 가중치

### 예측 궤적

예측 궤적을 입력 열 $U$ 에 대해 선형으로 나타냅니다.

$$
Z = \Phi\, z_0 + \Psi\, U
$$

- $Z = [z_1^T,\, \ldots,\, z_N^T]^T$, $U = [u_0^T,\, \ldots,\, u_{N-1}^T]^T$  
- $\Phi$, $\Psi$ 는 $A_d$, $B_d$ 로부터 구성합니다.

### QP 형태

$J$ 를 $U$ 에 대한 2차식으로 정리하면 다음 QP가 됩니다.

$$
\min_U \ \frac{1}{2}\, U^T H\, U + c^T U
$$

$$
H = 2\,(\Psi^T \bar{Q}\, \Psi + \bar{R}), \qquad c = 2\, \Psi^T \bar{Q}\, (\Phi\, z_0 - Z_{ref})
$$

- $\bar{Q} = \mathrm{blkdiag}(Q,\ldots,Q,Q_N)$, $\bar{R} = \mathrm{blkdiag}(R,\ldots,R)$  
- 해: $U^* = -H^{-1} c$. **적용하는 것은 첫 입력만**: $\tau = G(q_r) + u_0^*$.

## 2. 수식–코드 매칭 (현재 구현: LQR+적분)

| 수식/동작 | 코드 |
|-----------|------|
| 선형화 $A$, $B$ | `dynamics.py`: `linearize(Q_REF)` |
| LQR 이득 $K$ (CARE) | `run.py`: `solve_care(A, B, Q_mpc, R_mpc)` → `K_mpc` |
| 적분 $\int(q - q_r)\,dt$ | `run.py`: `int_q += (q - Q_REF) * DT`, anti-windup clip |
| $u = -K(z - z_r) - K_i\,\mathrm{int\_q}$ | `run.py`: `u0 = -K_mpc @ (z - Z_REF) - K_i * int_q` |
| $\tau = G(q_r) + u$ | `run.py`: `tau = G(Q_REF) + u0` |

## 4. 입·출력, 제약, 초기조건

| 구분 | 내용 |
|------|------|
| **입력** | 현재 상태 $z$. 목표 $z_r$ (상수). |
| **출력** | $\tau = G(q_r) + u$ (LQR + 적분 보정) |
| **제약** | 토크 사후 클리핑 $\|\tau_i\| \le 50$ N·m. 적분 anti-windup. |
| **초기조건** | $q(0)=[0,0,0]^T$, $\dot{q}(0)=0$. 목표 $q_r=[0.5,\,0.4,\,0.3]^T$. |

## 5. 외란 실험

$t \in [1.5,\,2.5]$ 초 동안 $\tau_{dist} = [5,\,-2,\,1]^T$ N·m 를 가합니다.

LQR+적분이 외란에 대해 목표로 다시 수렴하는지, 무외란 궤적과 비교할 수 있습니다.
