# 로봇 팔 제어 (3-DOF 평면 로봇)

이 프로젝트는 **관절 3개**를 가진 평면 로봇 팔을 제어하는 예제 모음입니다.  
동역학 기반 **PID**, **임피던스**, **LQR**, **MPC**, **QP** 다섯 가지 제어기를 각각 별도 폴더에 구현해 두었습니다.

각 폴더에는 다음이 들어 있습니다.

- 제어 이론 **수식** (가독성 있게 정리)
- **수식과 코드의 대응** 표
- **입력·출력·제약·초기조건** 설명
- **외란 실험** (무외란 vs 유외란 비교)
- **결과 그림** (`*_result.png`)

코드는 학습용으로, 함수와 변수에 대한 **주석**이 달려 있습니다.

## 로봇 모델

- **3-DOF 평면 로봇**: 회전 관절 3개, 2D 평면에서만 움직입니다.
- **동역학**은 모든 폴더의 `dynamics.py`에서 같은 식을 사용합니다.

$$
M(q)\,\ddot{q} + C(q,\dot{q}) + G(q) = \tau
$$

- $M(q)$: 관성 행렬  
- $C(q,\dot{q})$: 코리올리스·원심력  
- $G(q)$: 중력  
- $\tau$: 관절에 가하는 토크

## 폴더 구조 및 실행 방법

| 폴더 | 설명 | 실행 |
|------|------|------|
| [pid/](pid/) | 동역학 기반 PID (Computed Torque) | `cd pid && python run.py` |
| [impedance/](impedance/) | 작업공간 임피던스 제어 | `cd impedance && python run.py` |
| [lqr/](lqr/) | LQR (평형점 선형화) | `cd lqr && python run.py` |
| [mpc/](mpc/) | MPC 스타일 (LQR+적분으로 목표 추종) | `cd mpc && python run.py` |
| [qp/](qp/) | QP 기반 목표 각도 추종 | `cd qp && python run.py` |

각 폴더는 **독립적으로** 실행할 수 있습니다. 해당 폴더로 이동한 뒤 `python run.py`만 실행하면 됩니다.

## 입·출력, 제약, 초기조건, 외란

- **입력·출력·제약·초기조건**: 각 제어기 폴더의 README에 있는 **「입·출력, 제약, 초기조건」** 표를 보세요.
- **외란 실험**: 모든 `run.py`는 **외란 없음**과 **외란 있음** 두 경우를 한 번에 그립니다.
  - 외란 구간: $t \in [1.5,\,2.5]$ 초
  - 외란: 관절 토크에 $\tau_{dist} = [5,\,-2,\,1]^T$ N·m 를 더해 줌
  - 실선: 무외란, 점선: 유외란, 붉은 영역: 외란 구간
