# 임피던스 제어 (Impedance Control)

**작업공간**(엔드이펙터 위치)에서 목표 위치 $x_d$ 에 대해, 마치 **가상의 스프링·댐퍼·질량**이 붙어 있는 것처럼 동작하도록 토크를 계산하는 제어입니다.  
관절각이 아니라 **엔드이펙터의 위치·강성·댐핑**을 다루고 싶을 때 사용합니다.

---

## 이 제어기를 쓰는 이유

- **작업공간 제어**: 사람과 협업하거나 환경과 접촉할 때는 “관절각”보다 **손끝(엔드이펙터) 위치**와 **얼마나 뻑뻑/부드럽게 반응할지**가 중요합니다. 임피던스 제어는 목표 위치 $x_d$ 주변에 **원하는 강성 $K_d$, 댐핑 $B_d$, 질량 $M_d$** 를 가상으로 부여합니다.
- **접촉 시**: 외력이 가해지면 EE가 밀려났다가, 스프링·댐퍼 특성에 따라 부드럽게 돌아옵니다. 힘 제어가 아니라 **위치 + 임피던스** 로 동작해 구현이 상대적으로 단순합니다.
- **구현 방식**: “원하는 EE 가속도” $\ddot{x}_{des}$ 를 임피던스 법칙으로 정한 뒤, **역기구학(가속도 수준)** 으로 $\ddot{q}_{des}$ 를 구하고, $\tau = M\,\ddot{q}_{des} + C + G$ 로 토크를 계산합니다.

---

## 1. 수식 정리

### 작업공간과의 관계

- $x = f_{kin}(q)$: 정기구학 (엔드이펙터 위치)
- 속도·가속도는 자코비안 $J(q)$ 로 연결됩니다.

$$
\dot{x} = J(q)\,\dot{q}
$$

$$
\ddot{x} = J(q)\,\ddot{q} + \dot{J}(q,\dot{q})\,\dot{q}
$$

### 임피던스 법칙 (가상의 힘)

목표와 현재의 위치·속도·가속도 차이에 스프링·댐퍼·질량을 곱한 **가상의 힘** $F$ 를 정의합니다.

$$
F = M_d\,(\ddot{x}_d - \ddot{x}) + B_d\,(\dot{x}_d - \dot{x}) + K_d\,(x_d - x)
$$

- $M_d,\, B_d,\, K_d$: 목표 질량·댐핑·강성 행렬 (보통 대각 행렬로 설정)

### 원하는 EE 가속도

위 법칙에서 원하는 엔드이펙터 가속도를 다음처럼 둡니다.

$$
\ddot{x}_{des} = \ddot{x}_d + M_d^{-1}\Big[ B_d\,(\dot{x}_d - \dot{x}) + K_d\,(x_d - x) \Big]
$$

### 관절 가속도로 바꾸기

$\ddot{x}_{des} = J\,\ddot{q}_{des} + \dot{J}\,\dot{q}$ 이므로, $\ddot{q}_{des}$ 는 유사역행렬 $J^\dagger$ 로 구합니다.

$$
\ddot{q}_{des} = J^\dagger\,\big( \ddot{x}_{des} - \dot{J}\,\dot{q} \big)
$$

### 토크 계산

동역학을 이용해 위 가속도를 만들기 위한 토크를 계산합니다.

$$
\tau = M(q)\,\ddot{q}_{des} + C(q,\dot{q}) + G(q)
$$

## 2. 수식–코드 매칭

| 수식 | 코드 |
|------|------|
| $x = f_{kin}(q)$ | `dynamics.py`: `forward_kinematics(q)` |
| $\dot{x} = J\dot{q}$ | `run.py`: `xd = jacobian(q) @ qd` |
| $J(q)$ | `dynamics.py`: `jacobian(q)` (2×3) |
| $\dot{J}\dot{q}$ | `dynamics.py`: `jacobian_dot(q, qd)` |
| $\ddot{x}_{des} = \ddot{x}_d + M_d^{-1}[\cdots]$ | `run.py`: `xdd_des = XDD_D + np.linalg.solve(M_d, ...)` |
| $\ddot{q}_{des} = J^\dagger(\ddot{x}_{des} - \dot{J}\dot{q})$ | `run.py`: `qdd_des = J_pinv @ (xdd_des - J_dot @ qd)` |
| $\tau = M\,\ddot{q}_{des} + C + G$ | `run.py`: `tau = M(q) @ qdd_des + C_vec(...) + G(q)` |

## 4. 입·출력, 제약, 초기조건

| 구분 | 내용 |
|------|------|
| **입력** | 목표 EE 위치 $x_d$, 속도 $\dot{x}_d$, 가속도 $\ddot{x}_d$. 현재 $q$, $\dot{q}$. |
| **출력** | 관절 토크 $\tau \in \mathbb{R}^3$ |
| **제약** | 토크 제한 $\|\tau_i\| \le 50$ N·m |
| **초기조건** | $q(0)=[0.3,\,0.2,\,0.1]^T$, $\dot{q}(0)=0$. 목표 $x_d=[0.6,\,0.3]^T$ m. |

---

## 5. 외란 실험

$t \in [1.5,\,2.5]$ 초 동안 $\tau_{dist} = [5,\,-2,\,1]^T$ N·m 를 가합니다.

외란 구간에서 EE 위치가 벗어났다가, **댐핑 $B_d$** 덕분에 다시 목표 근처로 수렴하는 것을 확인할 수 있습니다.
